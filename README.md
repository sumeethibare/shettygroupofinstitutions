## Getting Started

First, run the development server:

```bash
npm run dev
```

after running the script you can view the project on browser by this url

```bash
https://localhost:3000
```

Tools Used for creation
NextJs,
React Icons,
Tailwind CSS,
Daisy UI,
Vercel,
Farmer Motion,
GSAP & Many More.
